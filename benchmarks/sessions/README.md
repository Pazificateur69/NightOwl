# Benchmark Sessions

Store one file per benchmark run in this directory.

Suggested naming:

- `YYYY-MM-DD-dvwa.md`
- `YYYY-MM-DD-juice-shop.md`
- `YYYY-MM-DD-webgoat.md`

Rules:

- one target per file
- one commit or tag per session
- record both detections and misses
- keep raw notes even if the results are bad

Start from [../results-template.md](../results-template.md).
